# Source migration

Migration is the process of moving anime/manga between sources without losing progress. This is most often used when a source is no longer accessible or another source is more up-to-date.

Always make sure to have a backup in case anything unexpected occurs.

Downloaded episode(s)/chapter(s) do not transfer with migrations.

Migrations with downloaded episode(s)/chapter(s) may leave the download behind. You will need to remove these manually with a file manager.

# Migrating on Stable/Preview

Migrating from anime/manga

videocam
Click for video guide

  1. Tap into a Anime/Manga you would like to migrate.
  2. Go to
    more_vert
    Overflow     and tap Migrate .

    Aniyomi will do a global search of all the sources you have installed and enabled.

  3. Select the Source that you'd like to migrate to by tapping the Anime/Manga thumbnail.
  4. Choose which data you want to transfer over, and you're done.

Migrating from Source

videocam
Click for video guide

  1. Tap into
    explore
    Browse     on the bottom navbar.
  2. Press the Migrate Anime / Migrate Manga tab at the top right side of Manga Extensions tab.
  3. Select the Source that you'd like to migrate from.
  4. Tap the Anime/Manga you'd like to migrate from the Source.

    Aniyomi will do a global search of all the sources you have installed and enabled.

  5. Select the Source you'd like to migrate to by tapping the Anime/Manga thumbnail.
  6. Choose which data you want to transfer over, and you're done.

Searching only pinned sources
